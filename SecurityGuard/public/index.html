How It Works

What Is Already Built

  • Camera video capture directly in the browser.
  • Motion detection based on frame difference.
  • Direction detection: left, right, up, down.
  • Forward and backward estimation using motion area changes.
  • A separate sound for every motion type.
  • An alarm siren when the object moves back.
  • Overlay box and live status on top of the video.
  • Event log and adjustable sensitivity.
